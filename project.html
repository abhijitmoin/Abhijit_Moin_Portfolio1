Portfolio Website

About Project:

This is a personal portfolio website designed to showcase my skills, projects, and experience in web development. It features a clean and modern design with responsive layout to look great on all devices. Built using HTML, CSS, and JavaScript, the site includes sections like About Me, Projects, Skills, and Contact.

Technologies Used:

HTML5, CSS3, JavaScript

Challenges:

Creating a fully responsive design that works seamlessly on both desktop and mobile was a key challenge. I also focused on optimizing loading speed and smooth navigation.

What I Learned:

Improved my skills in responsive web design, CSS Flexbox/Grid, and enhanced user experience techniques.
